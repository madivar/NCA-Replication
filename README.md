# NCA Replication: ESE and Revenue

This repository contains a reproducible replication of a Necessary Condition Analysis (NCA)
examining Entrepreneurial Self-Efficacy (ESE) as a necessary condition for firm revenue.

## Structure

- `code/`
  - `NCA_Replication_Final4.R` – full analysis script
- `data/`
  - `04_final_nca_dataset_STRICT.csv` – final dataset used for NCA (STRICT specification)
- `output/`
  - Scatter plot and ceiling plots used for evaluation

## Notes
- No raw survey data or identifiers are included.
- Revenue is coded as log10 of the upper bound of the reported category.
- ESE is the sum of 21 task mastery items coded 0–3.
